Mapping Earthquakes

Creation of a webpage that hosts earthquake data and allows the user to toggle back and forth between overlays to display tectonic plate outlines and earthquake displays based on magnitude.

Github Repo

Project Overview

The purpose of this project is to visually show the difference in magnitudes of earthquakes across the world in the last seven days. GeoJSON was used to pull earthquake data from the US Geological Survey. This dataset contained data for the coordinates and magnitudes of earthquakes from the last seven days. Javascript and D3.js were used to build the scripts. Leaflet library was then used to plot the data on a Mapbox style map through an API request. The map is interactive and holds several layers to allow the user to toggle between map styles and whether or not to display earthquake information.
