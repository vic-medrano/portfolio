Cryptocurrencies

Dataset on cryptocurrency was processed and the principal components were reduced using PCA. Unsupervised machine learning was then applied to create a classification system for cryptocurrencies on the current trading market

Github Repo

Project Overview

The purpose of this project was to gather information for clients wanting to dive into the world of cryptocurrencies and understand which cryptocurrencies may be the most beneficial to invest their time, money, and effort into. The data was first cleaned and preprocessed to create a list of current and tradable currencies. Principal Component Analysis (PCA) was used to reduce the original 98 columns down to three principle components. A K-means algorithm was then applied to determine how many clusters would be needed to divide the current data. This was accomplished with the creation of an elbow curve to determine the appropriate amount of clusters.

The cryptocurrencies were then able to be classified by group according to their features using the unsupervised machine learning model.
